    Velkommen til våres nettside. Denne nettsiden kan gi deg tips på hvordan man kan lage en nettside. Vi legger ut artikkler på hvordan man kan bruke HTML, JavaScript og CSS.

    Les mer


    HTML

    I HTML lærer du deg å skrive med "h1-h6" overskrifter, avsnitt "p", lister "ul", "ol" og "li", lenker "a" og bilder "img". Du vil også bli lært hvordan du bruker de semantisk HTML som feks "header", "main", "section" og "footer".

    JavaScript

    I JavaScript vil du lære hvodan man manipulerer DOM (Document Object Model), men du vil også lære hvordan du bruker objekter, arrayer, click og input.

    CSS

    I CSS vil du lære om farger, fonter, bakgrunner, border, margin og padding. Du vil også lære bruken av Flexbox og Grid.
