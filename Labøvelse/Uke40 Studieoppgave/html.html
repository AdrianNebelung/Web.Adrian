    På denne siden vil du lære om HTML og hvordan du skal skrive det. Vi kommer til å gå igjennom trinn for trinn hvordan man bygger opp en HTML-struktur.



    HTML

    Alle HTML-sider starter med en grunnleggende struktur:

    <!doctype html >
    <html lang = "no">
    <head>
    <meta charset ="UTF-8">
    <title>Min første nettside</title>
    </head>
    <body>
    <h1>Hei, verden!</h1>
    <p>Dette er min første avsnitt i HTML.</p>
    </body>
    </html>

    Nøkkelord:
    !doctype html
    html
    head
    body

    !doctype

    html

    head

    body
